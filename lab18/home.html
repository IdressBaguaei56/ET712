Welcome to Node.js Server Example

This page is being served by a Node.js HTTP server using the fs.createReadStream and pipe() methods.

Features Demonstrated:

  • Custom module usage with require()
  • HTTP server creation with the built-in http module
  • File streaming with fs.createReadStream
  • Stream piping with pipe()

The server is running on port 4000. You're seeing this page because you've successfully connected to the server!
